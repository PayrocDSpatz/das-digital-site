# DAS Digital Landing Page

This is a simple **Next.js + TailwindCSS** site.

## Run locally

1) Install Node.js (LTS)
2) In this folder, run:

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

- Push this repo to GitHub
- Import it into Vercel
- Add your domain (dasdigitalai.com)
